# Social-Media-War
A 2-player game coded entirely in python 

Requirements:
  Python: 3.10 or above
  Pygame: 2.3.0 or above

Controls:
  Player 1:
    LCTRL: shoot bullets
    WASD: movement in 2D
  Player 2:
    RCTRL: shoot bullets
    Arrow keys: movement in 2D

Note:
  If not working, try to change the first 5 variable strings in "socmedwar.py" to the exact location of your folder installed in your machine.
  Example: 'C:Users/HP/social-media-war/hit.mp3'
